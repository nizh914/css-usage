# CSS Usage Data

### About
Prototype of displaying data about CSS usage across the web.

### Prerequisites
1. [nodejs](https://nodejs.org/) 
2. grunt
```
npm install -g grunt-cli
```

### Clone and build
Clone css usage
```
git clone https://github.com/nizh914/css-usage.git
```

Build it


```
cd css usage
```

```
npm install
```


```
grunt
```

Open index.html in your favorite browser
